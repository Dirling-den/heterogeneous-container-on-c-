# heterogeneous-container-on-c-

Здесь используется карта std::unordered_map<std::type_index, ...>, где каждый тип хранит свой std::vector<T>.

    add<T>() добавляет значение нужного типа.

    get<T>() возвращает весь вектор этого типа.

    at<T>(i) даёт доступ по индексу.

    size<T>() показывает размер конкретного типа.
